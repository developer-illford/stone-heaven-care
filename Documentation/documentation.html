Getting Started with Everly

Everly is a modern, responsive HTML template that makes your website stand out. Here's what you need to know:

  • Built on Bootstrap 5.3 for responsiveness.
  • Customizable to fit your brand.
  • Easy to use with clean code structure.

This template is powered by HTML5, CSS3, and JavaScript.

Installation

To install Everly, download the .zip file and extract it. Open the index.html file in your browser to view the template.

File Structure

D) JavaScript

The template includes JavaScript for interactive features like fade in animations. Below is an example configuration for a fade in: 


    document.addEventListener("DOMContentLoaded", () => {
        const fadeInElements = document.querySelectorAll(
            ".fade-in-up, .fade-in-down, .fade-in-left, .fade-in-right"
        );

        const observer = new IntersectionObserver(
            (entries, observer) => {
                entries.forEach((entry) => {
                    if (entry.isIntersecting) {
                        entry.target.style.opacity = "1";
                        entry.target.style.animationPlayState = "running";
                        observer.unobserve(entry.target);
                    }
                });
            },
            {
                threshold: 0.2,
            }
        );

        fadeInElements.forEach((el) => {
            el.style.animationPlayState = "paused";
            observer.observe(el);
        });
